Flower Creator Documentation

About

This project utilizes the canvas drawing API for javascript as well as this concept of algorythmic botany to create flowers using the golden ratio. I added my own twist by adding customization for the starting divergence and padding as well as a rate of change for both that can be manipulated to make interesting creations. This project also uses local storage to save settings and highscores, the bulma css library, custom web components (the navbar, the header, and the footer), and importing data and functions from other pages to allow for ease of programming across multiple pages (such as using the same functions in free play and challenge).
